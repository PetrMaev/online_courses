# Проект веб-приложения онлайн курсов
## Описание:
**Данное приложение предполагает создание веб-приложения онлайн курсов.**
## Цель проекта:
Реализовать приложение, которое позволяет пользователям размещать свои полезные материалы или курсы.
## Установка:
1. Клонируйте репозиторий:
```python
https://github.com/PetrMaev/online_courses.git
```
```python
git@github.com:PetrMaev/online_courses.git
```
2. Установите зависимости:
```python
poetry install
```
## Наполнение данными базы данных:
Для наполнения данными базы данных используйте фикстуры и кастомные команды.
1. Кастомная команда для заполнения базы данных курсами и уроками:
```python
python manage.py add_data
```
2. Кастомная команда для создания суперпользователя:
```python
python manage.py csu
```
3. Кастомная команда для создания группы модераторов и добавления пользователя-модератора:
```python
python manage.py create_moderator
```
4. Кастомная команда для заполнения базы данных платежами:
```python
python manage.py add_payments
```
## Использование:
Для запуска программы введите в терминале следующую команду:
```python
python manage.py runserver
```
## Запуск приложения с использованием docker-compose
1. Для запуска проекта с docker-compose выполните следующую команду:
```python
docker-compose up --build
```
2. Для остановки работающих контейнеров используйте следующую команду:
```python
docker-compose down
```
4. Для просмотра логов и мониторинга работы контейнеров используйте команду:
```python
docker-compose logs
```
3. Для вывода списка всех контейнеров используйте команду:
```python
docker-compose ps
```
## В проекте реализованы:
1. CRUD операции с курсами на базе ViewSet
2. CRUD операции с уроками, пользователями и платежами на базе Generics
3. Регистрация пользователей
4. Авторизация
5. Валидация ссылок на видеоматериалы уроков 
6. Пагинация списка курсов и уроков
7. Возможности подписки на курс
8. Рассылка электронных писем подписчикам курса об обновлениях
